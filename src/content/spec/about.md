# About
刚踏入高中的我，带着对新阶段的“折腾”，在这里开启了。
我算不上什么技术大牛，只是个爱电脑的普通高中生。
这个博客没有复杂的定位，也没有高深的内容，纯粹是我的 “成长记录仪”：一边记录高中生活里的细碎日常；一边分享和电脑相关的技术点滴，不管是踩过的坑、学到的知识，还是偶然发现的实用工具，都想在这里留个痕迹。
如果你也喜欢电脑、愿意听一听高中生的日常碎碎念，欢迎常来逛逛，一起交流呀～
